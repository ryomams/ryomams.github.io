Tool Adaptability External Publishing Prerequisites Tutorialization Notes
Bitsy Unknown Allows export to HTML5. Internet Access, Modern Web-Browser (Ideally Chrome or Firefox) Externally links a "getting started" tutorial. Exporting to HTML 5 will require basic web literacy.
Flipnote Studio Allows export to GIF, MP4, and JPEG Image Sequence. Nintendo DSi with Flipnote Studio preinstalled Provides built-in tutorial and example projects. Stitching JPEG image sequence will require video editing. Target audience is children.
Flipnote Studio 3D Allows export to GIF, MP4, and JPEG Image Sequence. Nintendo 3DS with Flipnote Studio 3D preinstalled Provides built-in tutorial and example projects. Stitching JPEG image sequence will require video editing. Has more tools than previous edition, Flipnote Studio. Target Audience is children.
Scratch (v. 2.x+) Not natively supported, enforces first-party publishing. Internet Access, Modern Web-Browser (Ideally Chrome or Firefox) Extensive native tutorialization.
