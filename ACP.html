Accessible Tools That I Like :)

making stuff on computers is overwhelming and I can't find any good lists online so here's a sortable list of some tools I like in no particular order. Click on a header to sort by it

Type Tool Publishing Prerequisites Tutorialization Notes
Game Engine Bitsy Allows export to HTML5. Internet Access, Modern Web-Browser (Ideally Chrome or Firefox) Externally links a "getting started" tutorial. Exporting to HTML 5 will require basic web literacy.
Animation (2D) Flipnote Studio Allows export to GIF, MP4, and JPEG Image Sequence. Nintendo DSi with Flipnote Studio preinstalled Provides built-in tutorial and example projects. Stitching JPEG image sequence will require video editing. Target audience is children.
Animation (2D) Flipnote Studio 3D Allows export to GIF, MP4, and JPEG Image Sequence. Nintendo 3DS with Flipnote Studio 3D preinstalled Provides built-in tutorial and example projects. Stitching JPEG image sequence will require video editing. Has more tools than previous edition, Flipnote Studio. Target Audience is children.
Game Engine Scratch (v. 2.x+) Not natively supported, enforces first-party publishing. Internet Access, Modern Web-Browser (Ideally Chrome or Firefox) Extensive native tutorialization.
